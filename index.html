Sobre

Somos a Organix, uma empresa que busca uma ambiente para agricultura mais saudável, visando o rodízio de culturas.

Nossa empresa utiliza adubos naturais e também inseticidas biológicos para evitar o contaminamento dos funcionários como do plantio.

Fazendo uso dessas práticas, há uma contribuição para um solo mais saudável para todos, sendo capaz de atender as necessidades de cultivo sem comprometer as gerações futuras.

Soluções

Drone fertilizante

O drone fertilizante tem como objetivo mapear o terreno fazendo uso da sua autonomia como também fertilizar e distribuir água o suficiente para todo o plantio, visando o mínimo de desperdício.

Protótipo Esfera

O prototipo esfera tem como objedivo detectar a fertilidade, Matéria orgânica, umidade, temperatura, PH, bactéria e entre outros do plantio. Equipado com duas camaras de monitoramento para visualização do terreno

Nosso software

O nosso software é interligado entre os drones. Todos os drones que estão conectados ao sistema tem o objetivo de enviar os dados coletados de duas em duas horas para o software.
